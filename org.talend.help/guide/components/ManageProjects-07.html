Exporting items

You can export multiple items from the repository onto a directory or an archive file. Hence you have the possibility to export metadata information such as DB connection or Documentation along with your Job Script or your Business Model, for example.

  • On the Repository, select the items you want to export
  • For multiple select, press the Ctrl key and select the relevant items.

Note: In the case you want to export a database table metadata entry, make sure you select the whole DB connection, and not only the relevant table, as this would prevent the export process to complete correctly.

  • Right-click while maintaining the Ctrl key down and select Export items on the pop-up menu.
  • Then select the directory where you want to store the exported items. Alternatively, define the archive file where to compress the files for all selected items.
  • Click Finish.
