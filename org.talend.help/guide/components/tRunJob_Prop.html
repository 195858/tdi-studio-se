tRunJob

tRunJob Properties

Component family
System
Function
Executes the job called in the component's properties, in the frame of the context defined.
Purpose
tRunJob helps mastering complex job systems which need to execute one job after another.
Basic settings
Schema type and Edit Schema
A schema is a row description, i.e., it defines the number of fields that will be processed and passed on to the next component. The schema is either built-in or remote in the Repository.
Click Edit Schema to make changes to the schema. Note that if you make changes, the schema automatically becomes built-in.
 
 
Built-in: You create and store the schema locally for this component only. Related topic: Setting a built-in schema of Talend Open Studio User Guide
 
 
Repository: You have already created the schema and stored it in the Repository. You can reuse it in various projects and job flowcharts. Related topic: Setting a repository schema of Talend Open Studio User Guide
 
CopyChild Job Schema
Click to fetch the child Job schema.
 
Job
Select the job to be called in and processed. Make sure you already executed once the job called, beforehand, in order to ensure a smooth run through the tRunJob.
 
Version
Select the child Job version that you want to use.
 
Context
If you defined contexts and variables for the job to be run by the tRunJob, select the applicable context entry on the list.
 
Context Param
You can change the selected context parameters. Click the plus button to add the parameters as defined in the Context of the child job.
 
Die on child error
Untick this box to execute the parent Job even though there is an error when executing the child Job.
 
Transmit whole context
Tick this box to transfer all the context variables from the child Job to the parent Job.
 
Generate Code
Click the button to validate the context selection in the tRunJob and generate the relevant code.
Usage
This component can be used as a standalone job or can help clarifying complex job by avoiding having too many sub-jobs all together in one job.
