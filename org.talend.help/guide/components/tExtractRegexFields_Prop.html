tExtractRegexFields

tExtractRegexFields properties

Component family
Processing/Fields
Function
tExtractRegexFields generates multiple columns from a given column using regex matching.
Purpose
tExtractRegexFields allows to use regular expressions to extract data from a formatted string.
Basic settings
Field to split
Select on the list the field (column) to split.
 
Regex
Enter a regular expression according to the programming language you are using.
 
Schema type and Edit Schema
A schema is a row description, i.e., it defines the number of fields that will be processed and passed on to the next component. The schema is either built-in or remote in the Repository.
Click EditSchema to make changes to the schema. Note that if you make changes, the schema automatically becomes built-in.
Click Sync columns to retrieve the schema from the previous component connected to tExtractRegexFields.
Built-in: You create and store the schema locally for the component. Related topic: Setting a built-in schema of Talend Open Studio User Guide
Repository: You have already created and stored the schema in the Repository. You can reuse it in various projects and job flowcharts. Related topic: Setting a repository schema of Talend Open Studio User Guide
Usage
This component handles flow of data therefore it requires input and output components.
