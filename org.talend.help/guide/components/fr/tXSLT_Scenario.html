Scénario : Transformer un fichier XML en HTML à l'aide d'une feuille de style XSL

Ce scénario décrit un job appliquant une feuille de style XSL à un fichier XML et génère un fichier HTML en sortie.

    • Cliquez et déposez un composant tXSLT.
    • Dans la vue Component, paramétrez les propriétés du fichier XML à transformer, ici une liste de titres MP3 et leur artiste correspondant.
    • Paramétrez le chemin d'accès au fichier XSL pour appliquer les transformations souhaitées.
    • Dans cet exemple, nous souhaitons ajouter une image et appliquer une feuille de style pour créer un tableau HTML.
    • Pour finir, renseignez le chemin d'accès au fichier HTML de sortie.
    • Enregistrez le job et appuyez sur F6 pour l'exécuter. Ouvrez le fichier HTML dans un navigateur Web et vérifiez la sortie.
