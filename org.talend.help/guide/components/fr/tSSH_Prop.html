tSSH

Propriétés du tSSH

Famille de composant
System
Fonction
Le composant tSSH renvoie les données d'un ordinateur distant via le protocole Secure Shell défini.
Objectif
Le tSSH permet d'entrer en communication avec un serveur distant et de renvoyer les informations de sécurité.
Basic settings
Host
Adresse IP.
 
Port
Numéro du port d'écoute.
 
User
Informations d'authentification de l'utilisateur.
 
Public Key/Password
Sélectionnez l'option adéquate.
Si vous choisissez l'option Public Key, assurez-vous que la clé ait été ajoutée à l'agent ou qu'aucune passphrase (phrase secrète) ne soit demandée.
 
Password/Private Key
Password : Renseignez le mot de passe.
Private key : Sélectionnez votre clé privée.
 
Commands
Saisissez la commande permettant de récupérer l'information de l'ordinateur distant.
 
Use timeout/timeout in seconds
Définit la durée avant expiration. Un message notifiant l'expiration sera généré si l'actuel temps de réponse dépasse ce délai d'expiration.
Utilisation
Ce composant peut être utilisé en standalone.
Limitation
L'utilisation de ce composant est adaptée pour les systèmes de type Unix.
