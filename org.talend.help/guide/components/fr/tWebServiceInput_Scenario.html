Scénario : Extraire des images à l'aide d'un Webservice

Ce scénario décrit un job constitué de deux composants dont le but est d'utiliser une méthode de service Web et d'afficher le résultat en sortie dans une vue standard.

La méthode récupère une adresse URL complète sous forme de chaîne en entrée et retourne une liste d'images provenant d'une page Web donnée.

    • Cliquez et déposez les composants tWebServiceInput et tLogRow.
    • Dans la vue Component du composant tWebServiceInput, configurez les spécifications WSDL : End Point URI, WSDL et SOAPAction URI lorsque cela est nécessaire.
    • Si le service Web invoqué requiert des informations d'authentification, cochez la case Need authentication? et renseignez ces informations.
    • Dans le champ Method Name, saisissez le nom de la méthode tel que défini dans la description du service Web. Le nom et la casse de la méthode saisie doit correspondre parfaitement à la méthode du service Web demandé.
    • Puis, dans le champ return class, sélectionnez la classe retournée correspondant au type de valeur attendue.
    • Dans la zone Parameters, cliquez sur le bouton (+) pour ajouter un ligne au tableau.
    • Puis saisissez le nom exact des paramètres tel qu'attendu par la méthode.
    • Dans la colonne Value, saisissez l'adresse URL du site Web à partir duquel les images seront extraites.
    • La colonne Class est automatiquement renseignée avec la classe retournée sélectionnée juste avant.
    • Connectez le composant tWebServiceInput au composant de sortie standard, tLogRow.
    • Appuyez sur F6 pour exécuter le job.

Toutes les images extraites du site Web spécifié sont retournées dans la vue Run, sous la forme d'une liste d'adresses URL.
