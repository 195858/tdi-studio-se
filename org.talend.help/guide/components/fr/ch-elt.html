Composants ELT

Ce chapitre passe en revue les propriétés des principaux composants présents dans la famille ELT de la Palette de Talend Enterprise Studio.

La famille ELT regroupe les connecteurs de bases de données les plus courants, ainsi que des composants de traitement, tous dédiés au mode ELT dans lequel les SGBD cibles deviennent des outils de transformation.

Ce mode supporte les bases de données les plus courantes, comme Teradata, Oracle, Vertica, Netezza, Sybase, etc.
