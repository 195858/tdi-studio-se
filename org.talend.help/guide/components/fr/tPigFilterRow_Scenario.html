Scénario : Sélectionner des lignes de données d'une relation et stocker le résultat dans un fichier local

Ce scénario décrit un Job a trois composants sélectionnant des lignes de données qui correspondent à une condition de filtre d'une relation et stockant les résultats du filtre dans un fichier local.

    • Déposez les composants suivants de la Palette dans l'espace de modélisation graphique : tPigFilterRow, tPigLoad, tPigStoreResult.
    • Cliquez-droit sur le tPigLoad pour le relier au tPigFilterRow à l'aide d'un lien Row > Pig Combine.
    • Cliquez-droit sur le tPigFilterRow pour le connecter au tPigStoreResult à l'aide d'un lien Row > Pig Combine.
    • Double-cliquez sur le tPigload afin d'ouvrir sa vue Basic settings.
    • Cliquez sur le bouton [...] à côté du champ Edit schema pour ajouter des colonnes au schéma du composant tPigLoad.
    • Cliquez sur le bouton [+] pour ajouter les colonnes Name, Country et Age et cliquez OK pour sauvegarder la configuration.
    • Dans la liste Mode, sélectionnez Local.
    • Dans le champ Input filename, renseignez le chemin d'accès au fichier d'entrée.
      Dans ce scénario, le fichier d'entrée est CustomerList qui contient les lignes de noms, les noms des pays et les âges des clients.
    • Sélectionnez PigStorage dans la liste Load function.
    • Laissez les autres paramètres tels qu'ils sont
    • Double-cliquez sur le tPigFilterRow afin d'ouvrir sa vue Basic settings.
    • Cliquez sur Sync columns pour récupérer la structure du schéma du composant précédent.
    • Cochez la case Use advanced filter et saisissez, dans le champ Filter, l'expression :
      Country matches `PuertoRico'<
      Cette expression de filtre sélectionne les ligne de données contenant "PuertoRico" dans CustomerList.
    • Double-cliquez sur le composant tPigStoreResult pour ouvrir sa vue Basic settings.
    • Cliquez sur Sync columns pour récupérer la structure du schéma du composant précédent.
    • Dans le champ Result file, renseignez le chemin d'accès au fichier de résultat.
      Dans ce scénario, le résultat du filtre est sauvegardé dans le fichier Lucky_Customer.
    • Sélectionnez PigStorage dans la liste Store function.
    • Laissez les autres paramètres tels qu'il sont.
    • Sauvegardez votre Job et appuyez sur F6 pour l'exécuter.

Le fichier Lucky_Customer est généré et contient les lignes de données sélectionnées.
