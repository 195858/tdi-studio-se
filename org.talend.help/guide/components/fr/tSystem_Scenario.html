Scénario : Echo `Hello World!'

Ce scénario est constitué de deux jobs affichant un message dans la console de la vue Run.

    • Cliquez et déposez les composants tSystem et tPerl dans l'espace de modélisation.
    • Cliquez-droit sur le composant tSystem et sélectionnez un lien de type ThenRun pour relier ces deux composants. Lorsque vous exécutez le job, le premier composant déclenchera le deuxième.
    • Double-cliquez sur le composant tSystem et sélectionnez l'onglet Basic settings :
    • Dans le champ Command, saisissez la commande echo suivie de la chaîne de caractères "Hello World!" à afficher.
    • Dans le champ Output, sélectionnez l'option To a global variable pour envoyer la sortie dans une variable globale.
    • Puis sélectionnez le composant tPerl.
    • Saisissez la commande Perl permettant d'afficher la variable de sortie du tSystem dans la console.
    • Cliquez sur la vue Run et exécutez le job.

Le job exécute une commande echo et affiche la sortie dans la console de la vue Run grâce à la fonction Print de la commande Perl du composant tPerl.
