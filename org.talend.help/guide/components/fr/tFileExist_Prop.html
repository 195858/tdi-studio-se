tFileExist

Propriétés du tFileExist

Famille du composant
File/Management
Fonction
Le composant tFileExist vérifie si un fichier existe ou non.
Objectif
Le tFileExists aide à rationaliser les processus en automatisant les tâches récurrentes et fastidieuses, comme par exemple vérifier si un fichier existe ou non.
Basic settings
File Name
Chemin d'accès au fichier dont vous voulez vérifier l'existence.
Utilisation
Ce composant peut être utilisé en standalone.
Limitation
n/a
