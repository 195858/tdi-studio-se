Composants System

Ce chapitre passe en revue les principaux composants de la famille System de la Palette dans la perspective Integration du Studio Talend.

La famille System regroupe les composants permettant d’interagir avec votre système d’exploitation.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
