tJava

Propriétés du tJava

Famille de composant
Processing
Fonction
Le tJava transforme n'importe quelle donnée saisie en code Java.
Objectif
Le tJava est un éditeur (Java) et aussi un outil très flexible dans le job.
Basic settings
Code
Saisissez le code Java en fonction de la commande ou de la tâche que vous souhaitez exécuter. Pour plus d'informations concernant la syntaxe des fonctions Java, consultez l'aide en ligne de Talend Open Studio dans le menu Help Contents > Developer Guide > API Reference.
Utilisation
Ce composant est généralement utilisé pour débogger une erreur mais peut aussi être utlisé pour afficher le contenu d'une variable.
Limitation
Il est nécessaire de connaître le langage Java.
