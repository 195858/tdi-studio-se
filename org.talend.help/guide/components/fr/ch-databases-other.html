Databases - Autres composants

Le chapitre Databases - Autres composants décrit les connecteurs donnant accès à un éventail de bases de données et fournissant les outils de gestion de bases de données. Ces connecteurs couvrent divers besoins : ouverture de connexion, lecture et écriture de tables, commit de transactions, ainsi que des rollbacks pour le traitement des erreurs. Ces composants sont présents dans la famille Databases de la Palette de Talend Enterprise Studio.

D'autres types de connecteurs de bases de données, tels que les connecteurs de bases de données traditionnelles et de bases de données d'appliance, de datawarehouse, de bases de données verticales, sont documentés dans le Chapitre 7, Databases - Composants traditionnels et le Chapitre 8, Databases - Composants d'appliance et de datawarehouse.

Pour des informations concernant le support des versions des différentes bases de données, consultez l'annexe des versions des bases de données supportées, dans le Guide d'installation de votre Studio.
