Scénario : Retourner une réponse "Hello world"

Ce scénario comprend deux Jobs : un Job consommateur et un Job fournisseur.

Créer un Job fournisseur

Cette section explique comment créer un Job fournisseur.

      • Déposez les composants suivants de la Palette dans l'espace de modélisation graphique : un tESBProviderRequest, trois tLogRow, deux tJavaFlex et un tESBProviderResponse.
      • Connectez les composants comme dans la capture d'écran.
      • Configurez les propriétés du composant tESBProvider Request_1.
      • Configurez le schéma du composant tESBProvider Request_1.
      • Configurez les propriétés du composant tLogRow_1.
      • Configurez le schéma du composant tLogRow_1.
      • Configurez les propriétés du composant tJavaFlex_1. Ajouter le code suivant dans le champ Main code comme suit.
        Document requestTalendDoc = row2.payload;
        org.dom4j.Document requestDoc = requestTaldndDoc.getDocument();
        org.dom4j.Element rootElement = requestDoc.getRootElement();
        String text = rootElement.getTextTrim();
        System.out.println("### " + text);
        row3.content = text;
      • Configurez le schéma du composant tJavaFlex_1.
      • Configurez les propriétés du composant tLogRow_2.
      • Configurez le schéma du composant tLogRow_2.
      • Configurez les propriétés du composant tJavaFlex_2. Ajoutez le code dans le champ Main code comme suit.
        String name = row5 content
        String faultText = "Hello, " + name + "!";

        org.dom4j.Document faultDoc = org.dom4j.DocumentHelper.createDocument();
        faultDoc.addElement(
        new org.dom4j.QName("response",
              org.dom4j.Namespace.get("http://talend.org/esb/service/job"))
        ).addText(faultText);

        Document faultTalendDoc = new Document();
        faultTalendDoc.setDocument(faultDoc);
        row6.payload = faultTalendDoc;
      • Configurez le schéma du composant tJavaFlex_2.
      • Configurez les propriétés du composant tLogRow_3.
      • Configurez le schéma du composant tLogRow_3.
      • Configurez les propriétés du composant tESBProviderResponse_1.
      • Configurez le schéma du composant tESBProviderResponse_1.
      • Sauvegardez le Job.

Créer un Job consommateur

Cette section explique comment créer un Job consommateur.

      • Déposez les composants suivants de la Palette dans l'espace de modélisation graphique : un tESBConsumer, trois tLogRow, un JavaFlex et un tFixedFlowInput.
      • Connectez les composants comme dans la capture d'écran.
      • Configurez le schéma du composant tFixedFlowInput_1.
      • Configurez les propriétés du composant tFixedFlowInput_1.
      • Configurez les propriétés du composant tJavaFlex_1. Ajoutez le code suivant dans le champ Main code comme suit.
        org.dom4j.Document doc = org.dom4j.DocumentHelper.parseText(row1.payloadString);
        Document talendDoc = new Document();
        talendDoc.setDocument(doc);
        row2.payload = talendDoc;
      • Configurez le schéma du composant tJavaFlex_1.
      • Démarrez le Job fournisseur. Dans le log d'exécution vous pouvez voir :
        ...
        web service [endpoint: http://127.0.0.1:8088/esb/provider] published
        ...
      • Double-cliquez sur le composant tESBConsumer_1 afin de l'ouvrir.
      • Ouvrez Service Configuration et saisissez le chemin d'accès au WSDL : http://127.0.0.1.8088/esbprovider?WSDL
      • Cliquez sur le bouton afin de récupérer le nom du port et le nom de l'opération.
      • Configurez le schéma d'entrée, le schéma de réponse (response) et le schéma d'erreur (fault) du composant tESBComsumer_1.
      • Configurez les propriétés du composant tJavaFlex_3. Ajoutez le code suivant dans l'onglet Main code comme suit.

        if (null != row4.faultDetail) {
        row6.faultDetailString = row4.faultDetail.getDocument().asXML()
        }
      • Configurez le schéma du composant tJavaFlex_3.
      • Configurez les propriétés du composant tLogRow_2.
      • Configurez le schéma du composant tLogRow_2.
      • Configurez les propriétés du composant tLogRow_3.
      • Configurez le schéma du composant for the tLogRow_3.
      • Sauvegardez votre Job.
