tFileDelete

Propriétés du tFileDelete

Famille de composant
File/Management
Fonction
Le composant tFileDelete supprime un fichier d'un répertoire spécifié.
Objectif
Ce composant permet de rationnaliser les traitements en effectuant des tâches récurrentes et fastidieuses de manière automatique.
Basic settings
File Name
Chemin d'accès au fichier à supprimer.
Utilisation
Ce composant peut être utilisé en standalone.
Limitation
n/a
