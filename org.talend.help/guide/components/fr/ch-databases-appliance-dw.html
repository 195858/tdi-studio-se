Databases - Composants d'appliance et de datawarehouse

Le chapitre Databases - Composants d'appliance et de datawarehouse décrit les connecteurs de bases de données spécialisées dans le traitement de gros volumes de données. Ces connecteurs couvrent divers besoins : ouverture de connexion, lecture et écriture de tables, commit de transactions, ainsi que des rollbacks pour le traitement des erreurs. Ces composants sont présents dans la famille Databases de la Palette dans la perspective Integration du Studio Talend. D'autres types de connecteurs de bases de données, tels que les connecteurs des bases de données traditionnelles et de gestion de bases de données, sont documentés dans le Chapitre 7, Databases - Composants traditionnels et le Chapitre 9, Databases - Autres composants.

Pour obtenir des informations concernant le support des versions, consultez l'annexe des versions des bases de données/des systèmes tiers supporté(e)s, dans votre Guide d’installation et de migration Talend.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
