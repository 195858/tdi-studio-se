tAmazonOracleRollback

Propriétés du tAmazonOracleRollback

Ce composant est étroitement lié aux composants tAmazonOracleCommit et tAmazonOracleConnection. Ils sont généralement utilisés ensemble lors d’une transaction.

Famille de composant

Cloud/AmazonRDS/Oracle

 

Fonction

Le composant tAmazonOracleRollback annule la transaction dans une base de données connectée.

Objectif

Ce composant évite le commit de transaction involontaire.

Basic settings

Component list

Sélectionnez le composant AmazontOracleConnection dans la liste s’il y a plus d’une connexion dans votre Job.

 

Close Connection

Décochez cette case pour continuer à utiliser la connexion une fois que le composant a exécuté sa tâche.

Advanced settings

tStatCatcher Statistics

Cochez cette case pour collecter les données de log au niveau du composant.

Utilisation

Ce composant est généralement utilisé avec d’autres composants AmazonOracle, notamment les composants tAmazonOracleConnection et tAmazonOracleCommit.

Limitation

n/a

Scénario associé

Ce composant est étroitement lié aux composants tAmazonOracleConnection et tAmazonOracleCommit. Ils sont généralement utilisés ensemble lors d’une transaction.

Pour un scénario associé au composant tAmazonOracleRollback, consultez la section intitulée « tMysqlRollback ».
