Scénario : Envoyer un email lors d'une erreur

Ce scénario est un job composé de trois composants permettant d'envoyer un email à des destinataires spécifiques lorsqu'une erreur se produit.

    • Cliquez et déposez les composants suivants dans l'espace de modélisation : tFileInputDelimited, tFileOutputXML, tSendMail.
    • Paramétrez les propriétés du tFileInputdelimited. Voir également : tFileInputDelimited.
    • Cliquez-droit sur le composant tFileInputDelimited et sélectionnez une connexion Row > Main. Glissez-la jusqu'au composant tFileOutputXML et relâchez lorsque le symbole de prise eléctrique apparaît.
    • Paramétrez les propriétés du tFileOutputXML.
    • Connectez le composant tFileDelimited au composant tSendMail via une connexion Run on Error.
    • Paramétrez les propriétés du composant tSendMail :
    • Saisissez les adresses email des destinataires et de l'expéditeur, ainsi que l'objet du message.
    • Saisissez le message contenant le code d'erreur produit à l'aide de la variable globale correspondante. Accédez à la liste des variables en appuyant sur Ctrl+Espace.
    • Ajoutez des pièces jointes et des informations sur les en-têtes, si nécessaire. Saisissez les informations SMTP.

Dans ce scénario, le fichier contenant les données à transférer vers une sortie XML ne peut être trouvé. Le composant tSendmail s'exécute lorsque cette erreur est rencontrée et envoie un mail de notification au destinataire spécifié.
