Composants FileScale

Cette famille de composants est dépréciée mais reste disponible dans le studio.

Ce chapitre passe en revue les principaux composants présents dans le groupe FileScale de la Palette dans la perspective Integration du Studio Talend. Néanmoins, ces composants n’apparaissent dans la Palette que si vous avez souscrit à Talend Big Data Studio.

La famille FileScale regroupe tous les composants dédiés au mode ELT permettant de traiter très rapidement de gros volumes de données contenus dans de très gros fichiers.

Les composants FileScale utilisent le lien FS Combine afin de connecter des composants entre eux et de combiner tous les processus de transformation de données en un seul, dans le but d'optimiser les performances.

Pour utiliser les composants de cette famille, vous devez installer l'exécutable FileScale. Ce fichier, cependant, n'est plus fourni. Les composants FileScale sont donc fournis uniquement pour les clients ayant précédemment installé cet exécutable. Les nouveaux clients ne peuvent demander ce fichier binaire.
