Composants FileScale

Ce chapitre passe en revue les principaux composants présents dans le groupe FileScale de la Palette de Talend Enterprise Studio. Néanmoins, ces composants n’apparaissent dans la Palette que si vous avez souscrit à Talend Enterprise Data Integration Big Data edition.

La famille FileScale regroupe tous les composants dédiés au mode ELT permettant de traiter très rapidement de gros volumes de données contenus dans de très gros fichiers.

Cette famille ne sera dorénavant pas maintenue mais reste disponible dans le studio. Si toutefois vous devez utilisez ses composants, assurez-vous d'installer l'exécutable FileScale nécessaire à leur utilisation:

  1. Dézippez le fichier archive sur la/les même(s) machine(s) client que celle(s) sur laquelle/lesquelles vous avez installé le Studio.

  2. Dézippez le fichier .tar.

    Note

    Le dossier DOC contient le manuel utilisateur de FileScale, ainsi que d’autres informations (exemples, références, etc.).

  3. Ouvrez le répertoire correspondant au type d’architecture de votre serveur.

  4. Dans ce répertoire, créez un fichier (sans extension) nommé “license” contenant uniquement votre clé de licence.

    Note

    Lors de l’installation de FileScale sous Windows, copiez aussi le sous-répertoire contenant les fichiers manifest (présent dans le dossier win32-i386).

Le programme FileScale ne peut être exécuté que sur les architectures des systèmes d’exploitation Windows et Linux :

  • win32-i386 pour Windows 32/64 bits (Windows 2000 ou ultérieur),

  • linux_elf-i386 pour Linux 32 bits (Linux 2.6.18 ou ultérieur),

  • linux-amd64 pour Linux 64 bits (Linux 2.6.18 ou ultérieur).
