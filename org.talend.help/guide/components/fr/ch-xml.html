Composants XML

Ce chapitre passe en revue les propriétés des principaux composants présents dans la famille XML de la Palette de Talend Enterprise Studio.

La famille XML regroupe les composants dédiés aux tâches liées au XML, notamment le parsing, la validation, la création de structures XML, etc.
