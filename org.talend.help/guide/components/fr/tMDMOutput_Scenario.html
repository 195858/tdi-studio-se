Scénario : Ecrire des données dans une base de données MDM

Ce scénario décrit un Job à deux composants qui génère des données, les transforme en XML et les charge dans la base de données MDM.

    • A partir de la Palette, glissez les composants tFixedFlowInput de la famille Misc, et tMDMOutput de la famille Talend MDM dans l'espace de modélisation.
    • Reliez-les via des liens de type Row Main.
    • Double-cliquez sur le composant tFixedFlowInput pour afficher l'onglet Basic settings de la vue Component et paramétrer ses propriétés.
    • Dans la liste Schema Type, sélectionnez Built-in et cliquez sur le bouton [...] à côté du champ Edit schema pour décrire la structure des données que vous souhaitez charger dans la base de données MDM.
    • Ajoutez trois colonnes de type String en cliquant sur le bouton [+]. Nommez ces colonnes ISO2Code, Name, Currency. Cliquez sur OK.
    • Dans le champ Number of rows, saisissez le nombre de lignes que vous souhaitez générer. Ici, nous souhaitons générer qu'une seule ligne.
    • Dans la zone Mode, sélectionnez le mode Use Single Table pour ne générer qu'un seul jeu de données.
    • Dans le tableau Values, saisissez entre guillemets, dans chaque champ Value, la valeur correspondant à chaque colonne du schéma.
    • Dans l'espace de modélisation, double-cliquez sur le composant tMDMOutput pour afficher l'onglet Basic settings de la vue Component et paramétrer ses propriétés.
    • Dans la liste Property Type, sélectionnez Built-in et renseignez les champs manuellement. Si vous avez stocké les informations de connexion au MDM dans les métadonnées du Repository, sélectionnez Repository dans la liste et les champs suivants seront renseignés automatiquement.
    • Dans la liste Schema Type, sélectionnez Built-in et cliquez sur le bouton [...] à côté du champ Edit schema pour décrire la structure des données que vous souhaitez charger dans la base de données MDM.
    • Dans la zone de droite, ajoutez une colonne de type String en cliquant sur le bouton [+]. Nommez cette colonne xmlOutput. Cliquez sur OK.
    • La liste XML Field est automatiquement renseignée avec la colonne de sortie que vous venez de créer.
    • Dans le champ URL, saisissez, entre guillemets, l'URL de connexion à la base de données MDM.
    • Dans les champs Username et Password, saisissez l'identifiant et le mot de passe de connexion à la base de données.
    • Dans le champ Version, saisissez entre guillemets le nom de la perspective à laquelle vous avez accès. Laissez le champ vide pour afficher la perspective par défaut.
    • Dans le champ Data Model, saisissez entre guillemets, le nom du modèle dans lequel vous souhaitez écrire.
    • Dans le champ Data Container, saisissez, entre guillemets, le nom du répertoire de sauvegarde dans lequel vous souhaitez écrire.
    • Dans la vue Component, cliquez sur l'onglet Advanced settings pour paramétrer les paramètres avancées du composant.
    • Cliquez sur le bouton [...] du champ Configure Xml Tree pour ouvrir l'éditeur du composant.
    • Dans la zone Link target à droite, cliquez dans le champ Xml Tree.
    • Sélectionnez le texte rootTag et remplacez-le par "Country".
    • Dans la zone Linker source, sélectionnez les trois colonnes de votre schéma et glissez les sur le noeud Country.
    • Dans la boîte de dialogue [Selection], sélectionnez Create as sub-element of target node pour que vos trois colonnes correspondent à trois sous-éléments du noeud Country en XML. Cliquez sur OK
    • Cliquez-droit sur l'élément Currency dans la zone Link target, et sélectionnez Set as Loop Element dans le menu.
    • Cliquez sur OK pour fermer l'éditeur.
    • Enregistrez votre Job et appuyez sur F6 pour l'exécuter.

Le nouveau pays créé est ajouté à la base de données MDM.
