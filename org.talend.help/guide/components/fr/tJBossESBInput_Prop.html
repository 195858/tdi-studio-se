tJBossESBInput

Propriétés du tJBossESBInput

Famille de composant
Internet
Fonction
Le composant tJBossESBInput permet de récupérer un message d'un serveur JBossESB afin de le traiter comme un flux pouvant être utilisé dans un Job Talend.
Objectif
Le composant tJBossESBInput permet de passer un message JBossESB à un Job Talend.
Basic settings
Message key
Saisissez la clé du message à récupérer et à transformer en un flux.
Advanced settings
tStatCatcher Statistics
Cochez cette case pour collecter les données de log au niveau du Job ainsi qu'au niveau de chaque composant.
Utilisation
Ce composant est utilisé en standalone. Il doit être utilisé avec un lien de type OnSubjobOk vers un tJava.
Limitation
n/a
