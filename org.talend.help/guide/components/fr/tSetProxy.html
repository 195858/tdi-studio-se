tSetProxy

Propriétés du tSetProxy

Famille de composant

Internet

 

Fonction

Le composant tSetProxy est conçu pour configurer un proxy.

Objectif

Le tSetProxy vous permet de saisir les informations de configuration de votre proxy.

Basic settings

Proxy type

Liste des protocoles de proxy.

 

Proxy host

Adresse du serveur du proxy.

 

Proxy port

Numéro du port du proxy.

 

Proxy user

Identifiant d'authentification pour le proxy.

 

Proxy password

Mot de passe d'authentification pour le proxy.

 

None proxy hosts

Les hôtes auxquels vous devez vous connecter directement et non via le serveur de proxy. La valeur peut être une liste d'hôtes, séparés par le caractère |. Le caractère de remplacement (*) peut être utilisé pour la mise en correspondance, par exemple : "192.168.0.* | localhost".

Advanced settings

tStatCatcher Statistics

Cochez cette case pour collecter les données de log au niveau du Job ainsi qu'au niveau de chaque composant.

Utilisation

Le tSetProxy peut être utilisé en sous-job indépendant, mais peut également être déployé avec d'autres composants de la famille Internet.

Limitation

n/a

Scénario associé

Aucun scénario n'est disponible pour ce composant.
