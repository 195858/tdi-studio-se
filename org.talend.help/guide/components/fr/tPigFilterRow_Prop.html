tPigFilterRow

Propriétés du tPigFilterRow

Famille de composant
Processing/Fields
Fonction
Le composant tPigFilterRow vous permet de sélectionner une ou plusieurs ligne(s) de données à partir d'une relation basée sur des conditions définies.
Objectif
Ce composant sélectionne des données ou filtre les données à partir d'une relationr basée sur des conditions définies de filtre.
Basic settings
Schema et Edit Schema
Un schéma est une description de lignes, i.e., il définit le nombre de champs qui sont traités et passés au composant suivant. Le schéma est soit local (built-in) soit distant dans le Repository.
 
 
Built-in : Le schéma sera créé et conservé pour ce composant seulement. Voir également : Paramétrer un schéma built-in du Guide utilisateur de Talend Integration Suite Studio.
 
 
Repository : Le schéma existe déjà et est stocké dans le Repository. Ainsi, il peut être réutilisé dans divers projets et Jobs. Voir également : Paramétrer un schéma du Repository du Guide utilisateur de Talend Integration Suite Studio.
 
Filter configuration
Cliquez sur le bouton Add sous la table Filter configuration pour ajouter une (des) ligne(s) et configurer une ou plusieurs condition(s) de filtre.

Cette table disparaît si vous cochez la case Use advanced filter.

 
Use advanced filter
Cochez cette case afin de définir les conditions avancées de filtre en saisissant une expression de filtre dans le champ Filter.
 
tStatCatcher Statistics
Cochez cette case pour collecter les données de log, aussi bien au niveau du Job qu'au niveau de chaque composant.
Utilisation
Ce composant est généralement utilisé comme étape intermédiaire. Il nécessite donc un composant d'entrée et un composant de sortie.
Limitation
La connaissance des scripts Pig est nécessaire.
