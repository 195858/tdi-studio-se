Composants Processing

Ce chapitre passe en revue les propriétés des principaux composants présents dans la famille Processing de la Palette dans la perspective Integration du Studio Talend.

La famille Processing regroupe les composants vous permettant d’effectuer différents types de traitement de flux de données, notamment l’agrégation, le mapping, la transformation, la dénormalisation, le filtrage, etc.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
