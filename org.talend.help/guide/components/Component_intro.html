Components

This chapter details the main components' properties provided in the Palette of Talend Open Studio.

About the Components
Each component has a specific list of properties and parameters, editable through the Basic and Advanced tabs of the Component view.
WARNING-The Component properties have been splitted into Basic and Advanced settings tabs. As the information provided below may have been written beforehand, discrepancies between Talend Open Studio GUI and the description below are to be foreseen.
In the component properties section, an icon or points out whether the component is available in Java and/or in Perl.
