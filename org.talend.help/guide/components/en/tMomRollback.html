tMomRollback

tMolRollback properties

This component is closely related to tMomCommit component. It usually does not make much sense to use these components independently in a transaction.

Component family

Internet

 

Function

tMomRollback rollbacks data from the MQ Server..

Purpose

Avoids involuntary commitment of part of a transaction.

Basic settings

Component list

Select the Connection component Used in your Job.

 

Close Connection

Clear this check box to continue to use the selected connection once the component has performed its task.

Advanced settings

tStatCatcher Statistics

Select this check box to collect log data at the component level.

Usage

This component is to be used along with Mom components, especially with tMomCommit.

Limitation

n/a
