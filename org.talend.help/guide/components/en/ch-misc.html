Misc group components

This chapter details the main components that you can find in Misc family of the Palette in the Integration perspective of the Talend Studio.

The Misc family gathers miscellaneous components covering needs such as the creation of sets of dummy data rows, buffering data or loading context variables.

Warning

For Talend Open Studio for Big Data, the Property type, Schema and Query Type of components are always Built-in. For how to edit a Built-in schema, see Talend Data Integration Studio User Guide.
