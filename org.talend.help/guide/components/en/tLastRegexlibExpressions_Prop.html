tLastRegexlibExpressions

This component does not appear in the Palette of the studio unless you have the right subscription to the right edition of Talend Data Quality.

tLastRegexlibExpressions

Component family
Data quality
Function
tRegexlibExpressions connects to a web service at http://regexlib.com/WebServices.asmx to get the N most recent expressions from the list of regular expressions.
Purpose
tFindRegexlibExpressions returns a dataset holding information about the N most recent regular expressions added to the library and that match the query at regexlib.com.
Basic settings
Schema type and Edit Schema
These fields are read-only. The output schema of this component contains the following fields: Title, Expression, Description, Matches, Non-matches, Author, Rating.
 
Max Rows
Enter the max number of the expressions you want to get.
Usage
This component is a start component. It requires an output flow, usually a csv file to be able later to import the collected expressions into Talend Data Quality studio.
For more information about importing patterns, see How to import patterns from a csv file in Talend Data Quality User Guide.
