tSqoopImport

This component will be available in the Palette of the studio on the condition that you have subscribed to the relevant edition of Talend Integration Suite MPx.

tSqoopImport Properties

Component family
File/Sqoop
Function
tSqoopImport calls Sqoop to transfer data from a relational database management system (RDBMS) such as MySQL or Oracle into the Hadoop Distributed File System (HDFS).
Purpose
tSqoopImport is used to define the arguments required by Sqoop for writing the data of your interest into HDFS.
Basic settings
Connection
Type in the JDBC URL used to connect to the database where the source data is stored.
 
User name and Password
Type in the authentification information used to connect to the source database.
 
Table Name
Type in the name of the table to be transferred into HDFS.
 
Append
Select this check box to append transferred data to an existing dataset in HDFS.
 
File format
Select a file format for the data to be transferred. By default, the file format is textfile, but you can as well choose the sequencefile format instead.
 
Specify Number of Mappers
Select this check box to precise the number of map tasks (parallel processes) used to perform the data transfer.
 
Sprecify Target Dir
Select this check box to type in the path to, or to browse to the target file which you want transfer the source data to in HDFS.
 
Compress
Select this check box to enable compression.
 
Print Log
Select this check box to activate the Verbose check box.
 
Verbose
Select this check box to print more information while working, for example, the debugging information.
Advanced settings
Direct
Select this check box to use the import fast path.
 
Specify columns
Select this check box to activate the column table where you can specify the data columns you want to transfer into HDFS.
 
Use WHERE clause
Select this check box to use a WHERE clause that controls the rows to be transferred. In the activated field, you can type in the conditon used to selected the data rows you want. For example, type in id > 400 to import only rows where the id column has a value greater than 400.
 
Use MySQL default delimiters
Select this check box to use MySQL's default delimiter set
 
Additional arguments
Complete this table to use an additional argument if you need one.
At each time, only one additional argument is allowed. Use of multiple additional arguments at the same time may lead to exception error..
 
tStatCatcher Statistics
Select this check box to collect log data at the component level.
Usage
This component is used standalone.
Connections
 
Outgoing links (from one component to another):
Trigger: Run if; On Subjob Ok; On Subjob Error.

Incoming links (from one component to another):
Row: Iterate;
Trigger: Run if; On Subjob Ok; On Subjob Error; On Component Ok; On Component Error

For further information regarding connections, see Connection types in the Talend Integration Suite Studio User Guide.
