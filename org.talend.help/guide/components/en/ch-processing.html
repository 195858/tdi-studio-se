Processing components

This chapter details the main components that you can find in Processing family of the Palette in the Integration perspective of the Talend Studio.

The Processing family gathers together components that help you to perform all types of processing tasks on data flows, including aggregation, mapping, transformation, denormalizing, filtering and so on.

Warning

For Talend Open Studio for Big Data, the Property type, Schema and Query Type of components are always Built-in. For how to edit a Built-in schema, see Talend Data Integration Studio User Guide.
