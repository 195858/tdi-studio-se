tPatternExtract

This component is available in the Palette of the studio on the condition that you have subscribed to the relevant edition of Talend Data Quality.

tPatternExtract properties

Component family
Data Quality
Function
tPatternExtract extracts from a column all data strings that match a given Java regular expression.
Purpose
tPatternExtract allows to output all data that match a given pattern. You can then implement any required operation on the extracted data.
Basic settings
Column to check
Select the column you want to analyze.
 
Pattern type
Select from the list the pattern you want to check the data against and then extract all data that match the selected pattern.
 
Schema and Edit Schema
A schema is a row description, i.e., it defines the number of fields that will be processed and passed on to the next component. The schema is either built-in or remote in the Repository.
 
 
Built-in: You create the schema and store it locally for this component only. Related topic: Setting a built-in schema of Talend Integration Suite Studio User Guide.
 
 
Repository: You have already created the schema and stored it in the Repository. You can reuse it in various projects and job designs. Related topic: Setting a Repository schema of Talend Integration Suite Studio User Guide.
Advanced settings
tStatCatcher Statistics
Select this check box to collect log data at the component level.
Usage
This component is an intermediary step. It requires an input flow as well as an output.
