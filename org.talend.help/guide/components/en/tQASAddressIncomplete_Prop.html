tQASAddressIncomplete properties

Component family
Data Quality
Function
tQASAddressIncomplete filters addresses. It iterates on each row and reads all input addresses against the QuickAddress data.
Purpose
tQASAddressIncomplete can give two output flows: Incomplete and Reject. Incomplete collects all addresses that match to a partially-complete address in the QuickAddress data. Reject collects all addresses that match to several possible results or do not match to deliverable results in the QuickAddress data.
