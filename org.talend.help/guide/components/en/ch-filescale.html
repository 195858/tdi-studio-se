FileScale components

This chapter details the main components that you can find in the FileScale family of the Palette in the Integration perspective of the Talend studio. However, these components will only appear in the Studio Palette on the condition that you have subscribed to Talend Big Data Studio.

The FileScale family groups together ELT-mode dedicated components that process large quantities of data in large scale files at high speed.

The FS components use the FS Combine link to connect to each other, in order to combine all data transformation processes in one go to optimize performance.

This family is deprecated but remains available in the Studio. If you need to use the components from this family, you have to install the required FileScale executable, provided on demand:

  1. Unzip the archive file on the same client machine(s) as the one you installed the Studio on.

  2. Unzip the .tar file.

    Note

    The DOC directory contains user manual and other data (example, benchmark, etc.).

  3. Open the directory corresponding to the architecture of your server.

  4. In this directory, create a file (without extension) named “license” containing your license key and only your license key.

Note

When installing the executable for windows, also copy subdirectory containing manifest files (in win32-i386).

The FileScale program can only be run on the following Windows and Linux operating system architecture:

  • win32-i386 for windows 32/64 bits (Windows 2000 or greater),

  • linux_elf-i386 for Linux 32 bits (Linux 2.6.18 or greater),

  • linux-amd64 for Linux 64 bits (Linux 2.6.18 or greater).
