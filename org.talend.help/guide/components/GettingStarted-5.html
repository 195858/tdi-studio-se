Connecting to the remote repository

Note: Before connecting for the first time to the remote repository, make sure you received a Login and Password from the Administrator who created your user account stored in Talend Administrator.

On the login page of Talend Open Studio, click on the three dot button to open the Remote Repository connection dialog box.

By default the repository is stored locally.

To access a remotely stored repository, configure the connection access.

  • Add a connection to the Connections list using the plus button
  • In the Name field, give a name to this connection that will be listed on the login window of Talend Open Studio.
  • Select Database as Repository
  • Then fill in the details to access the database where the repository is stored as provided by your Talend Administrator.
  • Fill in also your User details as received from the administrator also.
  • Click Check to verify your database login connections.
  • Click OK to validate and go back to Login window.

On the login window, the newly created connection is now offered on the list of connections offered.

  • Select the relevant connection on the list.
  • Click the Refresh button to update the Existing project list offered.
  • Select the relevant project on the list, if more than one project is made available to this user. The list of projects displayed on the list are the projects allocated to the user in Talend Administrator.
  • Click OK to launch the selected project in Talend studio.

    • Note: According to your user Role defined in Talend Administrator, the Create, Import and Demos features are enabled or disabled.

Talend Open Studio allows to share items eventhough you developed them on your local application. For more information about how to import existing project items, see Importing items in the Remote Repository.

For more information about how to share a project, see Working on a shared project.
