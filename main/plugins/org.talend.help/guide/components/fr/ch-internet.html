Composants Internet

Ce chapitre passe en revue les propriétés des principaux composants présents dans la famille Internet de la Palette dans la perspective Integration du Studio Talend.

La famille Internet regroupe les composants vous permettant d’accéder à du contenu stocké sur Internet, des services Web aux e-mails en passant par les flux RSS, le SCP, le MOM, le FTP, etc.

Avertissement

Dans Talend Open Studio for Big Data, les options Property type, Schema et Query Type des composants sont toujours en mode Built-in. Pour plus d'informations concernant l'édition d'un schéma Built-in, consultez le Guide utilisateur de Talend Data Integration Studio.
