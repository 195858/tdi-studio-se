tSetKerberosConfiguration

Propriétés du tSetKerberosConfiguration

Famille de composant

Internet

 

Fonction

Le composant tSetKerberosConfiguration est conçu pour configurer l'authentification via Kerberos afin d'améliorer la sécurité des communications réseau.

Pour plus d'informations concernant le protocole Kerberos, consultez http://www.kerberos.org (en anglais).

Objectif

Le tSetKerberosConfiguration vous permet de saisir les informations d'authentification via Kerberos.

Basic settings

KDC Server

Adresse du centre de distribution de clés (Key Distribution Center, KDC).

 

Realm

Nom du royaume (realm) de Kerberos.

 

Username and Password

Informations d'authentification de Kerberos.

Advanced settings

tStatCatcher Statistics

Cochez cette case pour rassembler les métadonnées de traitement du Job au niveau du Job et au niveau de chaque composant.

Utilisation

Ce composant est généralement utilisé en tant que sous-job et est utilisé avec le tSoap.

Limitation

n/a

Scénario associé

Aucun scénario n'est disponible pour ce composant.
