Data Quality components

This chapter details the main components that you can find in the Data Quality family of the Palette in the Integration perspective of the Talend Studio.

This chapter also details the components which are specific to Talend Data Quality, the functionalities of which are completely integrated within Talend Studio. However, these components will only be available in the Studio Palette on the condition that you have subscribed to the relevant edition of Talend Data Quality.

The Data Quality family comprises dedicated components that help you improve the quality of your data. These components covers various needs such as narrow down filtering the unique row, calculating CRC, finding data based on fuzzy matching, and so on.

Warning

For Talend Open Studio for Big Data, the Property type, Schema and Query Type of components are always Built-in. For how to edit a Built-in schema, see Talend Data Integration Studio User Guide.
