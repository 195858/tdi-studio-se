Big Data components

This chapter details the main components that you can find in Big Data family of the Palette.

Using those components, you can connect, in the unified development environment provided by the Studio, to the modules of the Hadoop distribution you are using and perform operations natively on the big data clusters.

The Big Data components create connections to various third-party tools used for transferring, storing or analyzing big data, such as Sqoop, MongoDB and BigQuery and help you quickly load, extract, transform and process large and diverse data sets.

Warning

For Talend Open Studio for Big Data, the Property type, Schema and Query Type of components are always Built-in. For how to edit a Built-in schema, see Talend Data Integration Studio User Guide.

Also, if you have any questions, concerns or general comments please take part in our product forums which can be found at: http://www.talendforge.org/forum/index.php

For the version support information, see the appendix on supported third-party system/database versions in the Talend Installation and Upgrade Guide of the Talend solution you are using.
