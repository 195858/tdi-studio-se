Technical components

This chapter details the components you can find in the Technical group of the Palette in the Integration perspective of the Talend Studio.

The Technical components are Java-oriented components that perform very technical actions such as loading data in memory (in small subset of information) and keep it to allow its reuse at various stage of the processing.

Warning

For Talend Open Studio for Big Data, the Property type, Schema and Query Type of components are always Built-in. For how to edit a Built-in schema, see Talend Data Integration Studio User Guide.
