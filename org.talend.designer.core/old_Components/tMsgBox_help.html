tMsgBox
Purpose Create a dialogbox containing a message.
You can specifies title, icon and buttons of the dialog box.
Plateform Win32
Version 0.101 (ALPHA)
Signature This component is signed by talend

Example : 
tMsgBox(
      Message =>  "We're currently running on a " . Win32::GetOSName() . " OS !",
      Title =>  'My OS...',
      Icon => MB_ICONINFORMATION,
      Buttons => 0 );
Parameters :
Name Documentation Req. ? Default value Type
Title The title of the message box. No 'Integration Studio' STRING
Message The message that will be displayed in the the message box.

Note : you can use standard IS variable in it. 		No 'Integration Studio Message from $caller !' STRING
Icon The icon which will be display in the left upper corner of the message box.
This parameter can take one of the following value :
  • MB_ICONSTOP "X" in a red circle
  • MB_ICONQUESTION question mark in a bubble
  • MB_ICONEXCLAMATION exclamation mark in a yellow triangle
  • MB_ICONINFORMATION "i" in a bubble
No MB_ICONINFORMATION ELT
Buttons The button(s) of the message box.
This parameter can take one of the following value :
  • OK (code 0)
  • OK and Cancel (code 1)
  • Abort, Retry, and Ignore (code 2)
  • Yes, No and Cancel (code 3)
  • Yes and No (code 4)
  • Retry and Cancel (code 5)
No OK INT
Returns :
Name Type Documentation
Error Code INTEGER The function returns the menu id of the selected push button:
  • 0 Error
  • 1 OK
  • 2 Cancel
  • 3 Abort
  • 4 Retry
  • 5 Ignore
  • 6 Yes
  • 7 No
Error Message STRING The error message corresponding to the error code.
